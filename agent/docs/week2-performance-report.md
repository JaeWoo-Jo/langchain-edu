### 실패 케이스 분석

#### "안녕!", "존재하지않는품목" — 채점 기준 문제 (에이전트 정상)

이 두 케이스는 **에이전트 자체는 정상 동작**했다.

- "안녕!" → 에이전트가 친근하게 인사 응답 → TaskCompletion 0.9점
- "존재하지않는품목" → 에이전트가 "해당 품목은 없다"고 정확히 안내 → TaskCompletion 1.0점 만점

GEval에서 감점된 이유는 evaluation_steps에 "가격 정보가 구체적인 숫자와 단위를 포함하는지"가 포함되어 있기 때문이다.
가격 질문이 아닌 케이스에도 이 기준이 적용되어 억울하게 감점당한 것.

**개선 방향**: GEval의 evaluation_steps를 카테고리별로 분기하거나, 일반 대화/에지 케이스용 별도 메트릭을 사용한다.

#### "감자 얼마야?" — 에이전트 에러

ES에 감자 데이터(4건)가 존재하지만 에이전트가 "처리 중 오류가 발생했습니다"를 반환했다.
도구 호출 과정에서의 에러로 추정되며, 추가 디버깅이 필요하다.
